# Cyclone-V-AVMM-DMA
A configurable FPGA DMA controller using AVMM interfaces and sending a receiving data to FPGA using FIFOs. Supports up to 16 channels, tested on PCIe 2.0 x4, sends DMA interrupts using MSIX, supports one external interrupt source per DMA channel, allows for custom CSRs connected to AVMM interface
